# Quiz App

A simple interactive Quiz Application built with **React**.  
This project allows users to answer multiple-choice questions and see their final score at the end of the quiz.

The goal of this project was to practice React fundamentals and work with several important hooks such as **useState**, **useEffect**, **useRef**, and **useCallback**.

## Live Demo
https://maksym-yaremko-quiz-app.netlify.app/

## Features

- Multiple choice quiz questions
- Score calculation at the end of the quiz
- Navigation between questions
- Timer functionality using React hooks
- Clean and responsive UI

## Technologies Used

- React
- JavaScript (ES6+)
- CSS

## What I Learned

While building this project I practiced:

- Managing component state with **useState**
- Handling side effects with **useEffect**
- Optimizing functions using **useCallback**
- Working with DOM references using **useRef**
- Structuring a small React application
- Handling user interactions and quiz logic
